Laborator 5

Exercițiul 50 - Read File

Scrieți un program care afișează conținutul unui fișier text în câmpul Memo.


        

        
    


    

        
Exercițiul 51 - Viewer

        
Scrieți un program care afișează conținutul unui fișier text în câmpul Memo, utilizând caseta de dialog standard Open File pentru a obține numele fișierului.

        

            
            

        

        
    


    

        
Exercițiul 52 - Weather Database

        
Scrieți un program care adaugă informații zilnice privind temperatura la baza de date Weather (fișier text). Utilizați componenta tkcalendar pentru a introduce data. Dacă fișierul de date nu există, programul trebuie să-l creeze.

        

            
            

        

        
    


    

        
Exercițiul 53 - Jocul 15

        
Scrieți un program "Jocul 15". La început, jetoanele (numere de la 1 la 15) sunt amestecate pe o grilă 4x4, iar jucătorul trebuie să le alinieze în ordine corectă.

        

            
            

        

        
    


    

        
Exercițiul 54 - Asamblează imaginea

        
Scrieți un program "Asamblează imaginea" – un analog al jocului "15", în care jucătorul mută fragmente ale unei imagini. Imaginea se încarcă din fișierul pic_l.bmp, aflat în același director cu executabilul.

        

            
            

        

        
    

    

    

        
Exercițiul 56 - Memory Test

        
Scrieți un program care evaluează capacitatea unui jucător de a memora numere. Programul afișează numerele pentru o perioadă scurtă (ex.: 1 secundă) și apoi solicită utilizatorului să le reintroducă.

        

            
            

        

        
    

    

        
Exercițiul 57 - Advanced Memory Test

        
Scrieți un program ce evaluează capacitatea unui jucător de a memora numere, cu dificultate progresivă (de la numere de 4 cifre la 5, 6, etc.). Se trece la nivelul următor dacă numărul de numere corecte atinge anumite praguri.

        

            
            

        

        
    


    

        
Exercițiul 58 - Pair Pics

        
Elaborați un program care implementează jocul "Pair Pics". Tabla de joc este împărțită în celule care ascund imagini. La clic, o celulă se deschide pentru a afișa imaginea, iar scopul este găsirea perechilor de imagini identice. Imaginile se încarcă dintr-un fișier mare, unde sunt dispuse ca un singur sprite.

        

            
            

        

        
    

    
    
    

        
Exercițiul 59 - Căutătorul de mine (Sapper)

        
Dezvoltați un program care să implementeze jocul „Căutătorul de mine”. Jocul implică deschiderea celulelor și marcarea minelelor cu stegulețe.

        

            
            

        

        
        
    


    

        
Exercițiul 60 - Ceas de alarmă

        
Scrieți un program „Ceas de alarmă”. După stabilirea orei alarmei și a mesajului de reamintire, fereastra dispare, iar la ora stabilită apare un mesaj însoțit de semnal sonor.

        

            
            

        

        
    


    

        
Exercițiul 62 - Testare cu comutator

        
Scrieți un program de testare ce selectează răspunsul corect cu ajutorul unui comutator. Testul, definit într-un fișier, conține o secvență de întrebări și răspunsuri alternative. Numele fișierului se dă din linia de comandă.

        

            
            

        

        
    


    

        
Exercițiul 63 - Testare cu ilustrație

        
Scrieți un program de testare în care o întrebare poate avea și o ilustrație, iar numărul variantelor de răspuns este între 2 și 4. Textul întrebărilor și structura fișierului de test sunt definite în documentație.
